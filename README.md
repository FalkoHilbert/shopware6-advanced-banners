Create layered, responsive banners & slideshows like you could do with Shopware 5 Digital Publishing.

## Features
 - Banner editor
 - Layered banners
 - Image, text, wysiwyg layers
 - Easily extendible
 - 'Advanced Banner' CMS element
 - 'Advanced Banner Slider' CMS element
